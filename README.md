# homecare

O projeto resolve o controle manual dos serviços e pagamentos de cuidadoras de homecare, substituindo planilhas por um sistema simples para organizar atendimentos, datas, valores e fechamento diário.

Funcionalidades principais:
- cadastro de cuidadoras;
- registro de atendimentos;
- filtro de atendimentos por data inicial e data final;
- resumo dos valores cadastrados;
- relatório em PDF organizado por data.
